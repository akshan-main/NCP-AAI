# Course Completion Quality Gate

This checklist determines if the course is complete. Answer every question honestly. All answers in Sections 1 through 5 must be YES before attempting the NCP-AAI exam.

---

## Section 1: NVIDIA Platform Interaction Verification

- [ ] Did you create an account on build.nvidia.com?
- [ ] Did you generate and use an NVIDIA_API_KEY?
- [ ] Did you make at least one successful call to an NVIDIA-hosted NIM endpoint?
- [ ] Did you use the build.nvidia.com web interface to explore a model?
- [ ] Did you call an NVIDIA-hosted endpoint from Python code?

---

## Section 2: NVIDIA Tooling Verification

- [ ] Did you install and run NeMo Agent Toolkit locally?
- [ ] Did you configure and execute at least 3 different NAT agent types?
- [ ] Did you launch a NAT server or interactive workflow?
- [ ] Did you use NAT evaluation capabilities?
- [ ] Did you use NAT observability/profiling?
- [ ] Did you use at least one NAT middleware feature?
- [ ] Did you use NAT memory with at least one backend?
- [ ] Did you register and use a custom NAT function?

---

## Section 3: NVIDIA Safety Stack Verification

- [ ] Did you install and run NeMo Guardrails?
- [ ] Did you write and test at least one Colang 2.0 rail?
- [ ] Did you use Guardrails with an NVIDIA-hosted model?
- [ ] Did you demonstrate at least one safety control (jailbreak / topic / PII / hallucination / tool control)?
- [ ] Did you complete at least one adversarial / red-team exercise?

---

## Section 4: NVIDIA Reference Architecture Verification

- [ ] Did you study at least one NVIDIA Blueprint in detail?
- [ ] Did you reproduce a meaningful subset of a Blueprint?
- [ ] Did you document what you reused vs adapted?

---

## Section 5: Integration Verification

- [ ] Did you complete at least one capstone project?
- [ ] Does your capstone include evaluation results?
- [ ] Does your capstone include safety controls?
- [ ] Does your capstone include observability?
- [ ] Does your capstone include a deployment plan?

---

## Section 6: Certification Readiness

- [ ] Can you explain all 10 exam domains at interview depth?
- [ ] Did you complete the Module 13 mock exam with 80%+ accuracy?
- [ ] Did you complete all 8 milestones?

---

## Scoring

**Sections 1-5:** If any answer is NO, go back and complete it. The course is not finished until every item in these sections is checked.

**Section 6:** Determines exam readiness. These items are the final confirmation that you are prepared to sit for the NCP-AAI certification exam.
